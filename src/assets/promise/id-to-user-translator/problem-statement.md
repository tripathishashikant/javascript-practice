# The Problem: "The ID to User Translator"

Maano tumhare paas User IDs ki ek array hai: [1, 2, 3].

Tumhe har ID ke liye ek API call karni hai: fetchUser(id).

## Your Task

- Ek function likho fetchAllUsers(idsArray).

- Ye function ek Array of IDs lega.

- Ye har ID ke liye fetchUser(id) call karega.

- Ye tabhi resolve hoga jab saare users ka data aa jaye.

- Final result ek Array of Objects hona chahiye.
