# Problem: Coin Toss

Create a Promise that resolves or rejects randomly based on a virtual coin toss.

## Your Task

- Use `Math.random()` to generate a number.
- If the number is greater than `0.5`, resolve with `Heads: Jeet gaye!`.
- Otherwise, reject with `Tails: Harr gaye!`.
- Handle both cases with `.then()` / `.catch()`.
