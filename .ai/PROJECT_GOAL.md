# Project Goal

This repository is a JavaScript learning project that is evolving into an interactive practice website.

## Primary Goal

Build a Vue 3 + Vite + TypeScript web application where learners can:

- browse JavaScript practice topics and exercises
- read problem statements and supporting notes
- edit starter code directly in the browser
- run code and inspect output in a practice playground
- reset back to the original exercise state when the page reloads or the session ends

## Product Direction

The app should become a focused JavaScript practice lab rather than a generic documentation site.

Key experience goals:

- keep the interface simple and welcoming for learning
- show exercises in a structured way by topic
- make practice interactive and low-friction
- avoid persisting edits by default so each session starts fresh
- use a glassmorphism-inspired visual style

## Technical Direction

Current stack and conventions:

- Vue 3
- Vite
- TypeScript
- SCSS
- SMACSS-style stylesheet organization
- BEM naming for component classes

## Guidance For AI Systems

When helping in this repository:

- preserve the project as a learning-oriented codebase
- prefer clear structure over clever abstractions
- keep the UI implementation incremental and easy to understand
- treat the practice playground as the main product direction
- avoid introducing unnecessary complexity before the editor, runner, and exercise flow are stable
